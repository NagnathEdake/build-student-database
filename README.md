# Student Database Management System

A PostgreSQL-based Student Database project built using Bash Shell Scripting and SQL.  
This project was created as part of the :contentReference[oaicite:0]{index=0} Relational Database Certification.

The project demonstrates database creation, CSV data import automation, relational database management, and SQL query operations.

---

## 🚀 Features

- Import CSV data into PostgreSQL
- Manage students, majors, and courses
- Execute complex SQL queries
- Handle table relationships using foreign keys
- Automate database operations using Bash scripting

---

## 🛠️ Technologies Used

- PostgreSQL
- SQL
- Bash Shell Scripting
